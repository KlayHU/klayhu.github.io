指针：

⚡️Go语言指针没有运算，只存在值传递

指针的定义：

var a int =3

var pa *int = &a

*pa = 3

fmt.Println(*pa,pa)

▶️打印结果是:3 0xc000000a088

但是多次运行发现打印的地址都不一样，这与初始化内存空间有关。

指针值传递代码

func swap(a,s *int)(int,int){

​ *s,*a = *a,*s

}

func main(){

​ a:=3,s:=5

​ swap(&a,&s)

​ fmt.Println(a,s)

}

·☑️打印结果是：5 , 3
